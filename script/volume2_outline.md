# 第二卷「巫妖大劫」分場大綱 v1.0（劇情聖經 / Single Source of Truth）

> 多個子代理「平行擴寫各場對白」共用的權威大綱。每位擴寫者必須：
> 1. 嚴格遵守自己場景的「開場狀態」與「結束狀態」，確保跟前後場景無縫銜接。
> 2. 維持角色語氣：小艾＝毒舌但重情義、此卷對天道的質疑更冷更銳；Jack＝理性工程師、面對大劫的無力與「救能救的」的堅持。
> 3. 把「科學道：觀測→建模→驗證→修正」「不改大勢、改小勢」「天命亦可被審計」自然織進對白，不說教。
> 4. 輸出 VN step 序列（見最底格式），只用大綱列出的既有/新素材檔名。

## 卷定位

接續第一卷結尾（小艾完成第一份天道審計、卷末浮現巫妖兩股龐大氣運對撞的前兆）。時間已推進——人族在 Jack 庇護下繁衍數代，**阿禾已長大成部落支柱**，石爺年邁。

本卷主線：**巫妖兩族爭奪洪荒主導權，大劫爆發（不周山崩、天傾地陷）→ Jack 無法逆轉大勢，只做減災：轉移人族火種、保存巫妖雙方文明資料、降低無辜傷亡 → 小艾確認大劫是天道「粗暴清算機制」，質疑升級、解鎖更高權限 → 大劫尾聲的道德抉擇 → 巫妖兩敗俱傷退場，人族得喘息，科學道雛形更清晰。**

核心情感：**大勢碾過時，個體的無力與不放棄。** 母題＝文明（包括敵人的文明）的火種，比勝負與功德更值得守。小艾的質疑從「天道有沒有更優解」推進到「天道把眾生當可損耗的變數，這套演算法該被審計」。

卷末鉤子：接第三卷（輪迴審計，后土化輪迴）。

## 七場分場 Beat（每場：開場狀態 / 內容重點 / 結束狀態 / 素材 / 是否分支）

### v2c1 —「山雨欲來」
- **開場**：承第一卷。人族繁衍數代，阿禾已長大。巫妖兩股氣運對撞日烈，天地頻現異象（血月、地動、靈氣紊亂）。
- **內容**：小艾以科學道建模，預測大劫的規模、走向與「不可逆性」——這是洪荒級的大勢，不是上一卷那種小部落危機。Jack 第一次清楚意識到：有些系統性災難，工程師也只能減災，不能阻止。阿禾（成長後）登場，已是能獨當一面的部落支柱，與 Jack 亦師亦親。石爺年邁，叮囑後事的伏筆。
- **結束**：確認大劫將至且無法逆轉，Jack 決定把目標從「阻止」改為「減災與保存」。
- **素材**：bg=bg_twilight/bg_primordial_land；sprite=jack/xiaoai_serious/xiaoai_neutral/ahe_grown/elder；phone=radar_alert（大劫氣運讀數）。
- **分支**：無。next→v2c2

### v2c2 —「夾縫中的人族」
- **開場**：人族聚居地正位於巫妖預期戰場的邊緣，將被波及。
- **內容**：Jack 與阿禾規劃人族存續方案。資源與時間有限，必須抉擇。**玩家選擇**：(A) **舉族南遷**——遠離戰場核心，但長途跋涉、老弱難行、前路未知；(B) **就地築防避難**——守住家園與既有積累，但離戰場太近、一旦波及則退無可退。小艾把兩條路的存活率模型冷靜攤開，但強調「這次沒有絕對安全的選項，只有取捨」。兩路在 v2c3 匯流（被捲入大劫），但會被記成不同的處境。
- **結束**：人族依選擇做出部署，硬著頭皮迎接大劫。
- **素材**：bg=bg_migration/bg_tribe_day；sprite=jack/xiaoai_serious/ahe_grown；phone=radar_alert。
- **分支**：有（A 南遷 / B 築防），兩路 next 都→v2c3

### v2c3 —「大劫降臨」
- **開場**：巫妖正式全面開戰。
- **內容**：天崩地裂的總爆發——祖巫（zuwu）與妖族大能（yaodi）的正面碰撞撼動洪荒，不周山在大戰中崩折（天傾地陷、洪水傾瀉）。慘烈的災變鏡頭。人族被捲入邊緣，Jack 帶著阿禾在災中疏散求存，因果雷達紅光滿載（首次觸發 calamity_alert）。Jack 與小艾近距離見證巫妖大能的悲壯與毀滅——他們不是反派，是同樣被大勢推著走的存在。
- **結束**：大劫進入最慘烈階段，人族在夾縫中勉強存續，Jack 看清了「大勢清算」的真面目。
- **素材**：bg=bg_war/bg_buzhoushan；sprite=jack/xiaoai_shock/xiaoai_serious/zuwu/yaodi/ahe_grown；phone=calamity_alert、radar_alert。
- **分支**：無。next→v2c4

### v2c4 —「火種與資料」
- **開場**：大劫橫掃，巫妖文明與無數生靈正被抹去。
- **內容**：Jack 不去搶大劫的功德（那是大勢紅利），只做兩件他認為對的事：① 轉移、護住人族的火種與知識載體；② **保存巫妖雙方的文明資料**——他認為敵人的文明也是文明，不該隨大劫一起湮滅（小艾以「文明資料庫」ui_codex 記錄巫妖的語言、圖騰、技藝）。情感高潮：見證一位祖巫（或妖族大能）的殞落，對方在最後一刻察覺有人在「記住」他們的文明，神色複雜。小艾少見地沉默。
- **結束**：人族火種與巫妖文明資料被保下，Jack 在大劫中做了「無關勝負、只關乎文明」的選擇。
- **素材**：bg=bg_wasteland/bg_war；sprite=jack/xiaoai_neutral/xiaoai_serious/zuwu/yaodi；phone=merit_gain（保存之功的回饋）、ui_codex 文明資料庫。
- **分支**：無。next→v2c5

### v2c5 —「天道的演算法」
- **開場**：大劫尾聲，小艾握有全程數據。
- **內容**：小艾完成對大劫的完整建模，得出冰冷結論：這場大劫是天道的**「粗暴清算機制」**——當洪荒因果／氣運過載，天道便以一場大劫週期性「重置」，把累積的複雜度與衝突一次性燒掉。它有效，但**極度粗暴、把眾生當可損耗的變數**。小艾算出至少三種代價更低、犧牲更少的替代方案，但天道不採用——因為天道優化的是「系統穩定」，不是「個體存續」。質疑正式升級：天命不是真理，是一個沒被審計過、可能次優的演算法。因「保存文明、修補因果」之功，合道鎖再鬆一階，解鎖更高權限。
- **結束**：小艾的科學道更趨成形，Jack 立場更堅定：天道該被審計、被約束。
- **素材**：bg=bg_starfield_dao/bg_wasteland；sprite=jack/xiaoai_serious/xiaoai_smug；phone=patch_unlock、merit_gain。
- **分支**：無。next→v2c6

### v2c6 —「抉擇：救誰」
- **開場**：大劫餘波中，Jack 遇到一個具體而殘酷的道德工程抉擇。
- **內容**：時間、資源只夠救一邊——**玩家選擇（情感最重）**：(A) **保最多的性命**——集中力量搶救眼前最多能救的生靈（不分巫妖人），但承載文明記憶的載體（圖騰、典籍、火種器物）可能來不及帶走；(B) **保文明的火種**——拚死搶出能讓文明在他處重生的知識與火種，讓犧牲者「沒有白白消失」，但當下能救的活人更少。小艾不替他決定，只把兩種「代價模型」攤開——這正是科學道直面取捨、不逃避的內核。阿禾在此役以 Jack 教的方法獨當一面、做出關鍵貢獻。
- **結束**：抉擇落定，付出代價，但守住了所選的東西。兩路在 v2c7 匯流。
- **素材**：bg=bg_wasteland/bg_buzhoushan；sprite=jack/xiaoai_serious/xiaoai_neutral/ahe_grown；phone=calamity_alert、patch_unlock。
- **分支**：有（A 救人 / B 保火種），兩路 next 都→v2c7

### v2c7 —「殘陽下的審計」
- **開場**：大劫終於平息。巫妖兩族兩敗俱傷、元氣大傷退出洪荒主舞台，天地一片瘡痍，殘陽如血。
- **內容**：人族在廢墟與餘燼中倖存、得到喘息（暗示人族將承接洪荒主角的氣運，但 Jack 不為此沾沾自喜——那是無數犧牲換來的）。石爺安詳離世（若前文鋪墊），阿禾接過部落。小艾呈上她的**第二份天道審計報告**：大劫的清算邏輯、可避免的犧牲、天道的次優之處，條理分明、字字如刀。她說出本卷的題眼：「天道讓洪荒活下去——可它從沒問過，是不是非得死這麼多人。」Jack 望著殘陽下重新升起的人族炊煙，立志更堅。卷末，手機深處浮現新的因果線——一道與「死亡、輪迴、亡魂歸處」相關的龐大氣運正在匯聚（后土化輪迴的前兆）。next=null，旁白點題＋「第三卷『輪迴審計』製作中」。
- **結束**：巫妖退場、人族喘息、小艾科學道雛形清晰，輪迴的序幕將啟。
- **素材**：bg=bg_twilight/bg_wasteland；sprite=jack/xiaoai_serious/xiaoai_neutral/ahe_grown；phone=radar_alert（輪迴氣運前兆）、merit_gain。
- **分支**：無。next=null。卷末旁白點題＋「第三卷『輪迴審計』製作中」。

## step 格式（與 prologue.json / volume1.json 完全一致）

每個場景輸出為一個物件：
```
{
  "id": "場景id",
  "steps": [
    { "bg": "背景檔名" },
    { "sprite": "立繪檔名" },
    { "item": "道具檔名" },
    { "who": "角色key", "text": "對白或旁白" },
    { "phone": "事件名", "label": "手機面板顯示文字" },
    { "choice": [ { "text": "選項文字", "goto": "目標場景id" }, ... ] }
  ],
  "next": "下一場景id 或 null"
}
```
- 一個 step 可同時含 sprite + who + text。
- who 只能用：narration / jack / xiaoai / tiandao / power / nuwa / ahe / elder / zuwu / yaodi（ahe 現場戲用 ahe_grown 立繪；旁白用 narration）。
- 立繪檔名只能用：jack, xiaoai_neutral, xiaoai_smug, xiaoai_serious, xiaoai_shock, ahe_grown, ahe, elder, nuwa, zuwu, yaodi, silhouette_power。
- 背景檔名只能用：bg_war, bg_buzhoushan, bg_migration, bg_wasteland, bg_twilight, bg_starfield_dao, bg_primordial_land, bg_tribe_day, bg_tribe_night, bg_creation, bg_chaos。
- 道具/ui 檔名只能用：dao_phone, ui_radar, ui_cloak, ui_merit, ui_patch, ui_codex, ui_calamity。
- phone 事件名只能用：boot / radar_alert / cloak_on / merit_gain / patch_unlock / calamity_alert，並附 label。
- v2c2、v2c6 含玩家 choice（兩選項 goto 同一匯流場景）；v2c7 為卷末 next=null。
- 對白飽滿、有文學與畫面感，各角色語氣分明；每場對白量約為序章單場的 1.5–2 倍（較重場 v2c3/v2c4/v2c7 可更多）。
